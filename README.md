# KTB 3기 AI 해커톤 갤러리 (Kackerton Dashboard)

카카오 테크 부트캠프(KTB) 3기 수강생들이 진행한 **AI 해커톤 프로젝트를 모아보고 공유하는 갤러리 서비스**입니다.

모든 참가자가 자신의 프로젝트를 직접 등록하고, 서로의 프로젝트를 확인하며 좋아요를 누를 수 있습니다.

## 🚀 서비스 목적
- **프로젝트 공유**: 해커톤 결과물을 한눈에 볼 수 있는 아카이빙 공간 제공
- **상호 작용**: '조아요(Like)' 기능을 통한 팀 문항 격려 및 인기 투표
- **접근성**: 별도 회원가입 없이 누구나 등록 및 열람 가능 (수정은 비밀번호로 보호)

## ✨ 주요 기능

### 1. 프로젝트 등록 및 관리
- **간편 등록**: 팀 이름, 프로젝트명, 설명, URL 등 필수 정보만으로 빠르게 등록
- **자동 이미지 추출**: 서비스 URL 입력 시, `Microlink API`를 활용해 Open Graph 이미지를 자동으로 가져옵니다.
- **태그(Tag)**: 최대 3개의 기술 스택이나 키워드 태그를 등록할 수 있으며, 색상이 자동 지정된 뱃지로 표시됩니다.
- **수정 보안**: 프로젝트 등록 시 설정한 **숫자 4자리 비밀번호**를 통해 본인 팀만 수정할 수 있습니다.

### 2. 갤러리 및 인터랙션
- **그리드 뷰**: 반응형 카드 레이아웃으로 모든 프로젝트를 깔끔하게 볼 수 있습니다.
- **정렬(Sorting)**:
    - **최신순**: 가장 최근에 등록된 프로젝트부터 보기
    - **좋아요순**: 가장 인기가 많은 프로젝트 순으로 보기 (새로고침 시에도 유지됨)
- **좋아요(Like)**: 브라우저 세션 기반으로 중복 없이 프로젝트에 좋아요를 누를 수 있습니다.

### 3. UI/UX
- **반응형 디자인**: 모바일, 태블릿, 데스크탑 모두 최적화
- **토스트 알림**: 등록/수정 성공 시 부드러운 토스트 알림 제공
- **로딩 및 에러 처리**: 스켈레톤 UI 및 직관적인 에러 메시지

## 🛠 기술 스택
- **Frontend**: React, Vite
- **Styling**: Tailwind CSS, Framer Motion (Animation)
- **Backend/DB**: Firebase (Firestore)
- **Hosting**: Firebase Hosting

## 💻 실행 방법 (로컬 개발)

1. **의존성 설치**
   ```bash
   npm install
   ```

2. **개발 서버 실행**
   ```bash
   npm run dev
   ```

3. **빌드**
   ```bash
   npm run build
   ```

## 🚀 배포 방법

이 프로젝트는 Firebase Hosting을 사용합니다. 아래 명령어 중 하나로 빌드부터 배포까지 한 번에 수행할 수 있습니다.

### 방법 1 (NPM Script)
```bash
npm run deploy
```

### 방법 2 (Shell Script)
```bash
./deploy.sh
```

> **참고**: 배포를 위해서는 Firebase CLI가 설치되어 있고 로그인이 필요할 수 있습니다.
